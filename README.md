### 방식

  - 단순 책 내용을 타이핑하는 것이 아닌, **공부하며 새롭게 알게 된 내용** 위주로 정리합니다.

- **발표**: 당일 스터디 범위에 해당하는 챕터의 발표자는 **랜덤**으로 지정합니다.
